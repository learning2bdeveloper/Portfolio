GIRING - Tool for start-ups

Capstone Web & Mobile Project

A comprehensive tool designed to assist start-ups in managing their operations efficiently.The system includes a website for owners and a mobile application for employees to log in and input stocks, inventory, and expenses.

Key Functions:

  • Employee Tracking
  • Payroll w/ 2023 Tax Reduction
  • Bill Payment Reminder
  • Budgeting
  • Invoicing
  • Inventory
  • Expenditure Management
  • Analytics

Technologies Used: HTML, CSS, JavaScript, PHP, MySQL, Android Studio (Java)
